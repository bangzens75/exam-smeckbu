# Aplikasi Exambro SMK NU Bahrul Ulum

**Nama Sekolah:** SMK NU Bahrul Ulum  
**Alamat:** Jl. Raya Pelemwatu No.09 Menganti, Gresik  
**Motto:** "SMK Bisa dan Hebat"

## Deskripsi
Aplikasi ini digunakan untuk membuka tautan ujian berbasis web dengan aman.  
Pengguna dapat memindai kode QR berisi URL ujian, lalu aplikasi akan menampilkan halaman ujian di dalam WebView.  
Mode layar ganda (split-screen) dinonaktifkan untuk menjaga fokus ujian.

## Cara Build & Instalasi
1. Buka Android Studio.
2. Pilih **Open Project**, lalu arahkan ke folder `ExambroSMKNU`.
3. Tunggu hingga Gradle selesai melakukan sinkronisasi.
4. Jalankan proyek langsung ke perangkat atau bangun APK melalui menu:
   - *Build > Build Bundle(s) / APK(s) > Build APK(s)*
5. Instal APK ke perangkat Android.
6. Berikan izin kamera saat diminta untuk memindai kode QR.

## Fitur
- Pemindaian kode QR berisi URL ujian.
- Membuka halaman ujian di WebView internal.
- Tidak dapat digunakan dalam mode split-screen.
- Desain sederhana dan ramah pengguna.

## Catatan
- Pastikan koneksi internet stabil.
- Aplikasi ini hanya memindai tautan yang valid (http/https).
- Ganti ikon aplikasi dengan logo resmi sekolah telah diterapkan.
